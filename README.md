## Software para criar Diagrama de Classes

Existem vários, para download:

   - [StartUML](http://staruml.io/download)
   - [ArgoUML](http://argouml.tigris.org/)
   - [Dia](http://dia-installer.de/index.html.en)
   - Umbrello  
   ```
   apt-get install umbrello
   ```
   - [Astah](http://jude.change-vision.com/jude-web/index.html)
   - [Jude](http://jude.change-vision.com/jude-web/product/jude_pl.html)
   + Versões online:
     - [Genmymodel](https://www.genmymodel.com/)
     - [Draw.io](https://www.draw.io/)

> ArgoUML e Dia são bem simples de usar.
